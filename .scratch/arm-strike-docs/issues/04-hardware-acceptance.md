# Kinect Bridge 実機受入を行う

Status: `ready-for-human`

実装・自動試験完了後に実施する。

- 10分連続試験で平均25fps以上、UDP タイムアウトなしを確認する。
- 外部映像で入力反映の中央値と最大値を複数回測定し、目標100ms以内を確認する。
- 人物離脱・復帰、Bridge 再起動、Kinect 切断/再接続、Skeleton Stream 停止、障害継続時間と再起動後待機時間を確認する。
- Bridge再起動後10秒で復旧しない場合、Kinect連携利用不可・展示停止・技術担当への連絡を案内できることを確認する。
- 試験構成と結果を記録し、対応環境を更新する。
