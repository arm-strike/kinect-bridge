# Triage Labels

The engineering skills use five canonical triage roles. This file maps those roles to the label strings used in Kinect Bridge's local issue tracker.

| Canonical role | Label in this tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue. |
| `needs-info` | `needs-info` | Waiting for investigation or additional information. |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for Codex to work on. |
| `ready-for-human` | `ready-for-human` | Requires human confirmation. |
| `wontfix` | `wontfix` | Will not be actioned. |

When a skill mentions a role, use the corresponding label string from this table.
